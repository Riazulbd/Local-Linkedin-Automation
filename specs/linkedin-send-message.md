## LinkedIn Send Message Plan

1. Ensure profile page context is loaded.
2. Dismiss popups and messaging overlays.
3. Locate Message CTA from primary row selectors.
4. Click Message CTA and wait for composer.
5. Locate contenteditable textbox via selector priority list.
6. Human-type templated message content.
7. Re-check for blocking popups.
8. Locate and click Send button.
9. Verify message appears in thread and return success.

Fallbacks:
- button text selector
- aria-label selector
- conversation overlay textbox selectors
